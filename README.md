# JGEC_GYM_BROS


## Start the project

- clone the repo
#### Frontend start
1. cd frontend
2. npm i
3. npm run dev

#### Backend start
1. cd backend
2. npm i
3. npm install --save-dev nodemon
4. npm run dev
